# Title and Description Formula Reference

Quick reference for creating optimized title tags and meta descriptions based on page type.

## Title Tag Formulas

### 5 Proven Title Formulas

| Formula | Example | Best For |
|---------|---------|----------|
| `Keyword | Benefit | Brand` | SEO Guide | Improve Rankings | BrandName | General pages |
| `Number + Keyword + Promise` | 10 SEO Tips to Double Your Traffic | Listicles, guides |
| `How-to + Keyword` | How to Optimize Meta Tags for SEO | Tutorial pages |
| `Question + Keyword` | What is SEO? Complete Guide for Beginners | FAQ, informational |
| `Year + Keyword` | 2026 SEO Best Practices | Time-sensitive content |

### Title Length Guidelines

- **Optimal**: 50-60 characters
- **Maximum**: 60 characters (Google truncates after ~60)
- **Minimum**: 30 characters (too short = missed opportunity)

### Title Best Practices

1. Front-load primary keyword (first 5-10 chars)
2. Include brand name at end (after separator)
3. Use separators: `|`, `-`, `:`, `»`
4. Add power words: Complete, Ultimate, Best, Free, Guide, Tips
5. Include numbers when relevant (10, 5, 7, etc.)

## Meta Description Formulas

### Description Template

```
[Key Offer/Topic] + [Primary Benefit] + [Call to Action]
```

### Description Length Guidelines

- **Optimal**: 150-160 characters
- **Maximum**: 160 characters (Google truncates)
- **Minimum**: 120 characters (too short = incomplete message)

### Description Best Practices

1. Start with primary keyword/topic
2. Include unique value proposition
3. Add clear CTA (Learn more, Download, Get started, etc.)
4. Use emotional triggers where appropriate
5. Match the page's search intent

## Page-Type Templates

### Homepage

**Title**: `Brand Name | Short Value Proposition`
```
灵衢社区 | UnifiedBus互联协议社区
```

**Description**: `[Brand purpose] + [Key offerings] + [CTA]`
```
灵衢互联社区官网，提供规范文档下载、技术公开课、社区活动资讯。加入灵衢社区，共建算力互联生态。
```

### Documentation/Article

**Title**: `Article Topic | Complete Guide | Brand`
```
灵衢基础规范2.0 | 技术文档 | 灵衢社区
```

**Description**: `[What readers learn] + [Key topics covered] + [CTA]`
```
详解灵衢基础规范2.0修订版，包括系统组成、协议架构、编程模型。下载完整规范文档，掌握灵衢技术核心。
```

### Product/Software Page

**Title**: `Product Name | Key Benefit | Brand`
```
操作系统灵衢组件 | 算力池化软件 | 灵衢社区
```

**Description**: `[Product purpose] + [Key features] + [Download CTA]`
```
操作系统灵衢组件实现设备管理、内存池化、灵衢通信等核心功能。支持openEuler，免费下载源码。
```

### Category/List Page

**Title**: `Category Name | All [Items] | Brand`
```
软件资源获取 | 灵衢组件下载 | 灵衢社区
```

**Description**: `[What's available] + [How to access] + [CTA]`
```
下载灵衢操作系统组件、系统服务核心等软件资源。获取源码仓库、版本信息、安装指南。
```

### Legal/Policy Page

**Title**: `Page Name | Brand`
```
隐私声明 | 灵衢社区
```

**Description**: `[What the page covers] + [For whom]`
```
灵衢社区隐私声明，说明个人信息收集、使用、存储方式。了解我们如何保护您的隐私。
```

### Guide/Tutorial Page

**Title**: `How to [Action] | Guide | Brand`
```
灵衢厂商标识管理 | 操作指南 | 灵衢社区
```

**Description**: `[What the guide covers] + [Who it's for] + [CTA]`
```
灵衢厂商标识申请与管理完整流程。适用于硬件厂商、方案商，快速获取灵衢设备标识。
```

## CTR Optimization Elements

| Element | CTR Boost | Example |
|---------|-----------|---------|
| Numbers | +20-30% | "10 Tips", "5 Steps" |
| Year | +15-20% | "2026 Guide" |
| Power Words | +10-15% | "Ultimate", "Complete", "Best" |
| Questions | +10-15% | "How to", "What is" |
| Brackets/Parentheses | +10% | "(Free)", "[2026]" |

## Power Words List

### For Titles
- **Authority**: Ultimate, Complete, Definitive, Essential, Comprehensive
- **Value**: Free, Best, Top, Proven, Expert
- **Action**: How to, Learn, Discover, Master, Build
- **Urgency**: Now, Today, Quick, Fast, Easy

### For Descriptions
- **CTA**: Learn more, Download, Get started, Try free, Start now
- **Benefit**: Save time, Improve, Boost, Increase, Master
- **Emotional**: Discover, Unlock, Transform, Achieve
